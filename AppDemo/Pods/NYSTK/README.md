## <a id="How_to_use:"></a>How to use:
* Installation with CocoaPods：`pod 'NYSTK','~>0.0.1'`
* Manual import：
    * Drag All files in the `NYSMC` folder to project
    * Import the main file：`#import <NYSTK/NYSTK.h>`
